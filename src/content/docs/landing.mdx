# これだけは守ろう、やっちゃダメなこと！
<Aside type="danger" title="法律に反した書き込みはダメ！">
  ヘイトスピーチなど、がっつり法律で禁じられてるぞ！利用規約以前の問題だね。
</Aside>

- センシティブな文章は、そのまま LTL に流しちゃだめ！ちゃんと注釈を書いたうえで CW で隠すか、隔離をしよう！
- センシティブな画像や動画なんかは、そのままアップロードしちゃダメ！「センシティブフラグ」をつけよう！

<Card>
「センシティブ」ってのはえっちなものだけじゃなくて、政治的・宗教的にヤバいネタなんかも含むよ。

あと、「えっちなもの」っていうのもいわゆる R-18 だけじゃなくて、「人前で堂々と見られないもの」を指すよ。

イメージが湧かない人は、[misskey.io の NSFW の基準](https://support.misskey.io/hc/ja/articles/6657417016463-NSFW%E3%83%95%E3%83%A9%E3%82%B0-%E3%82%BB%E3%83%B3%E3%82%B7%E3%83%86%E3%82%A3%E3%83%96%E3%83%95%E3%83%A9%E3%82%B0-%E3%81%8C%E5%BF%85%E8%A6%81%E3%81%AA%E5%9F%BA%E6%BA%96%E3%81%AB%E3%81%A4%E3%81%84%E3%81%A6)なんかを参考にしてみるといいかも。(これはぼすきーの基準とは異なるから鵜呑みにするのは注意だけどね)

わかりやすい例えとしては、文章・画像ともに「町内会の掲示板に堂々と貼れないものは LTL に堂々と流すな」って覚えておくといいかも。

CW を使うときはちゃんと注釈を書こう！正しい注釈が書かれていないと、「開きたくない情報なのに判断できなくて開いちゃった」っていう人が出てくるぞ。

</Card>

- 「自主モデレーション」をしちゃダメ！

<Card>

- 規約に書いていない俺ルールを言うのはダメだぞ。
- 規約に書いてあることでも、battle になるような言い方はダメだぞ。
</Card>

あと、権利者本人が「この曲はこういう条件で使えるよ」って言ってて、それを守れる形でぼすきーに上げるのはいいんじゃないかな。
</Card>
